# EVA-01 Mobile UI Kit — Field Operator

iOS/Android-style prototype demonstrating the EVA-01 design system at phone scale.

## Screens
- **Home / OPS** — status badges, mission clock, KPI grid, subsystem progress, event log, FAB
- **Form / CMD** — operator directive form (text inputs, select, checkboxes)
- **Alerts** — alert stack and emergency response panel with PURGE hazard action
- **Auth** — operator profile

## Files
- `index.html` — entry
- `MobileApp.jsx` — all screens, custom angular phone shell, modal, tab bar, header

Imports `../web/components.jsx` to share Button, DataCard, ProgressBar, etc.

## Run
Open `index.html` in the preview pane.
